# Core Monorepo

This monorepo is a collection of packages used across multiple RabbyHub clients (e.g. [`Rabby`](https://github.com/RabbyHub/Rabby/), [`rabby-mobile`](https://github.com/RabbyHub/rabby-mobile/)).
